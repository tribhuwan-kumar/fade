# Fade brightness beyond level

_why?_

_sometimes the windows brightness level hurts my eyes even at 0 percent, so i wrote this application to fade the brightness to level zero i.e. total darkness, we can adjust that by a simple slider_


## Features:
- decrease brighness beyond default limit
- increase brightness as usual
- supports multiple monitors (experimental)
